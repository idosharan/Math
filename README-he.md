# אימון למבחן מתמטיקה — כיתה ג'

אפליקציית WEB קטנה (HTML + CSS + JS + JSON) שפועלת מקומית וללא אינטרנט. מותאמת גם למחשב וגם לנייד, בעברית ובכיוון כתיבה מימין לשמאל.

## איך מריצים?
1. הורידו את התיקייה או את הקובץ `kid-math-app.zip` ופרסו אותו.
2. פתחו את הקובץ `index.html` בדפדפן (Chrome/Edge/Firefox).
3. בחרו נושא, רמת קושי ומספר שאלות ולחצו **התחל**.

## נושאים כלולים
- לוח הכפל עד 10×10 (כולל שאלות "גורם חסר" ברמת מתקדם)
- סדר פעולות: סוגרים → כפל/חילוק → חיבור/חיסור (מבנים בטוחים ללא `eval`)
- תחום הרבבה (0..9,999): ערכי ספרות, עיגול, כתיבה מורחבת, והשוואת מספרים
- חיבור וחיסור עד 9,999 (התאמת טווח לפי רמה)
- חילוק: חילוק מדויק מלוח הכפל, וברמת מתקדם — מנה ושארית
- בעיות מילוליות מקובץ `problems.json`

## התאמה ועריכה
- הוספת/עריכת בעיות מילוליות: פתחו `problems.json` והוסיפו אובייקטים במבנה הקיים.
- שינוי עיצוב: ערכו את `styles.css` (צבעים, גדלים, כפתורים).
- שמירת ההישגים: האפליקציה שומרת שיאים מקומיים (LocalStorage) לפי נושא ורמה ומציגה "מדליות" בראש הדף.

## נגישות לנייד
- מקלדת מספרים על המסך (ניתנת לכיבוי)
- שדות קלט מסוג `tel` להצגת מקלדת מספרית בטלפון
- פריסה רספונסיבית

בהצלחה! 🌟
